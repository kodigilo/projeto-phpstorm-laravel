# Projeto em Laravel 8

## Recursos

- Docker (php7.3/mysql5.7)
- Laravel 8

```rash
1. composer install
2. cp .env.example .env
3. cp docker-compose-example.yml docker-compose.yml
3. docker-compose up -d
4. php artisan config:cache
5. php artisan config:clear
```
